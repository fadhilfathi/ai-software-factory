# AI Software Factory Frontend

This is the frontend component of the [AI Software Factory](..), a multi-agent platform for autonomous software development. Built with **Next.js 14+**, **TypeScript**, and **Tailwind CSS**.

It communicates with a **Go (Gin)** backend API.

## Getting Started

1. Ensure the backend and database services are running (via Docker Compose from the root or native development).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser.
